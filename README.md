# Slim in 6 健身 APP

## 痛点 Motivation

- 原版视频很经典, 但是无法编排
- 比如我想先做仰卧起坐, 多做两组俯卧撑, 最后才练腿, 当然还是要先热身
- ramp it up 中好几组深蹲是连着的, 每次做的时候都希望当中能穿插一段过渡动作休息一下
- 想以 slim in 6 中的燃脂运动为骨干, 能插入其他优秀哑铃动作视频进来
- 一直想能电子记录自己的运动日志

## APP 核心功能

- 将原版视频按做所的操作分割, 能够自主编排
  - 提供常用编排选项(预设, preset):
  - 记录用户的上一次编排
  - 针对每段给出一个运动积分, 分数以 start it up 总分 80 分, ramp it up 总分 100 分, burn it up 120 分 为基准分配
- 记录用户编排后的完成情况
- 在每段完成时有一个蒙版, 上面 继续(倒计时 2s) / 加休息 两个按钮, 按加休息可立刻插入一段过渡动作
- 日历视图, 报表视图, 查看自己每日的运动状况, 和各个动作被做的次数报表
- 运动完自拍一张, 还能记录体重, 作为"运动日记"
- Bounus 每日填表, 这个以后做:
  - 你今天是不是吃( )了:
    - 火锅
    - 曲奇
    - 蛋挞
    - 红烧肉
    - 油条
    - 超过两个肉丸
    - 超过两个包子
    - 超过 500ml 牛奶
  - 你今天的中饭热量:
    - 全家, 盒马可以算
  - 中午有去公园走圈吗
- 减肥核心 KPI: 体重减少量 / 运动总积分

## 界面

- 运动日记
  - 下半部分类似于朋友圈流视图, 自拍+体重
  - 上半部分是一个日历, 点日期进去可以看到
    - 当日运动成果
    - 表单, 填 Bounus 中的内容
- 编排界面
  - 最上面有一栏, 可横向滚动, 显示已编排的片段顺序, 总时长, 分数, 本栏 sticky top
    - 长按当中的片段可拖动, 做排序和删除
  - 列表 提供视频的预览图(动作帧), 分数, 点击即可加入
- 播放界面
  - 在每段结束时提供选项, 自动继续(倒计时两秒) / 休息一下(random select 过度动作)
  - 记录视频的完成情况, 拨了一半才可得一半分, 然后播完才可得全部分
